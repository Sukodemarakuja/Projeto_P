Esse projeto visa criar uma nova IA com base em outras IA's, fazendo com que ele seja um assistente pessoal único com o proprio banco de dados,
a ideia inicial era fazer um assistente que conseguisse conversar em tempo real com o usuario utilizando comandos de voz
essa ideia ainda há de ser implementada futuramente. Como o foco agora é conseguir ter uma base funcional todas as API que estão sendo utilizadas são gratuitas, o que faz o codigo ser de
facil acesso caso ele se torne publico um dia.
É bom destacar que esse codigo esta sendo feito com a ajuda do ChatGPT já que tenho bem pouco experiencia com programação (ele está AJUDANDO a fazer o codigo, não fazendo totalmente sozinho).
